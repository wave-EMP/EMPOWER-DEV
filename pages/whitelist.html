How To Use Polygon with Metamask

Automatic Setting RPC :
  • Install Metamask
  • Go to Browser => https://chainlist.org/
  • Search www.empowerdev.org
  • Search Network => Polygon, click “Polygon Mainnet”
  • Click add to Metamask
  • Then Approve and Switch Network
Manual Setting RPC :
  • Install Metamask
  • Open MetaMask and click the network dropdown menu, which is located at the top of the window.
  • Click “Custom RPC” and enter the details as seen in the screenshot below. We’ve also included a plain text version of the details below the screenshot so you can simply copy and paste.
    • Network Name: Polygon Mainnet New RPC
    • URL: https://rpc-mainnet.maticvigil.com/
    • Chain ID: 137
    • Currency Symbol (optional): MATIC Block
    • Explorer URL (optional): https://explorer.matic.network/
  • Click “save”. You’re now connected to the Polygon network!
How to Buy & Send Matic From Exchange to Metamask/TrustWallet :
  • Buy Matic From Exchange (Binance, KuCoin, TKO, etc.)
  • Send Matic using Polygon Network
  • Select Polygon Network then Confirm
